# Privacy Policy

**Last updated:** February 13, 2026

## Stock Price Predictor

This app is an educational tool that provides stock price predictions based on historical data. It does not collect, store, or share any personal information.

### Data Collection

- **No Personal Data:** This app does not collect any personal information from users
- **No Account Required:** Users do not need to create an account or provide any personal details
- **No Cookies:** This app does not use cookies
- **No Tracking:** We do not track user activity

### Data Usage

- The app fetches publicly available stock data from Yahoo Finance API
- Data is processed locally on the device for prediction purposes only
- No user data is transmitted to any third-party servers beyond Yahoo Finance

### Third-Party Services

This app uses the following third-party service:
- **Yahoo Finance** - Provides stock market data

### Changes to Privacy Policy

We may update this policy from time to time. Any changes will be reflected on this page.

### Contact

For any questions about this privacy policy, please contact the developer.
